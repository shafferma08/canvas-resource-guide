Where do I find student usernames and passwords?
You can find student credentials in Focus:
  1. Go to Network Account Info.
  2. Click Print a Form/Letter.
  3. Select Network Accounts.
  4. Look under Office 365 Account Credentials.
How do I reset a student's password?
You can request a reset in the Network Account Info screen in Focus:
  • Check the box Request Office365 Account Password Reset.
Note: This reset is NOT immediate. You must find out when the system processes it.

I'm getting a "Not a Tenant" error.
This error usually happens when you use the wrong login page. Start over and use the specific link above (ending in /login/saml/213). If you use a general Canvas link or the district link, it may not recognize your PTC account.
The login isn't accepting my email.
Make sure you are using your full @myptc.edu email address.
Do not use a personal email (Gmail, Yahoo) or a standard PCSB email.
Can Dual Enrollment students use this link?
No. Dual Enrollment students have a different login process and should not use this link.
